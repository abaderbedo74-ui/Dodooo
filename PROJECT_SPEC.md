# متجر الفساتين يدوية الصنع - المواصفات التقنية

## نظرة عامة على المشروع
متجر إلكتروني لبيع الفساتين يدوية الصنع مع نظام حجز والدفع آمن

## الصفحات المطلوبة
1. الصفحة الرئيسية (index.html)
2. المتجر - صفحة 1 (shop.html?page=1)
3. المتجر - صفحة 2 (shop.html?page=2)
4. المتجر - صفحة 3 (shop.html?page=3)
5. المتجر - صفحة 4 (shop.html?page=4)
6. صفحة تفاصيل المنتج (product.html)
7. صفحة العروض (offers.html)
8. صفحة اتصل بنا (contact.html)
9. صفحة تسجيل الدخول/تسجيل (login.html)
10. صفحة السلة والدفع (cart.html)

## المواصفات التقنية
- HTML5/CSS3/JavaScript (Vanilla)
- تصميم Responsive للموبايل والكمبيوتر
- تأثيرات Animations و Transitions
- نظام Dark/Light Theme
- Local Storage لحفظ بيانات العملاء
- نظام محاكاة للدفع آمن
